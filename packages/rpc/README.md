# Simple RPC mechanism.

Basic implementation uses `json_rpc_2`.

# Installation

pubspec.yaml

````yaml
dependencies:
  tekartik_rpc:
    git:
      url: https://github.com/tekartik/rpc.dart
      path: packages/rpc
    version: '>=0.1.0'
````

tekartik_rpc_test:
git:
url: https://github.com/tekartik/rpc.dart
